## Accessibilite

- Tous les boutons d'action ont un `aria-label` explicite
- Les badges de statut ont un `role="status"`
- Les modals utilisent `role="dialog"` avec `aria-modal="true"`
- Navigation clavier complete (Tab, Enter, Escape)
- Contraste minimum WCAG AA sur tous les textes
