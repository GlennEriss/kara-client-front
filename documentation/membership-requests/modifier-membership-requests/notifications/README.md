## Notifications – Modifier une demande d'adhésion

> **À compléter plus tard** : Cette section couvrira les notifications liées à la modification d'une demande d'adhésion.

### Notifications prévues

- **Toasts UI** :
  - Succès de modification.
  - Erreurs de validation ou de sauvegarde.

- **Notifications système** :
  - Notification aux admins après modification d'une demande (si nécessaire pour traçabilité).
  - Notification au demandeur si la modification impacte son dossier (selon stratégie).
