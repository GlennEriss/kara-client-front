## Diagrammes d’activité – Vue détails d’une demande d’adhésion

Objectif : décrire le flux utilisateur/administrateur depuis l’ouverture de la page détails (`MembershipRequestDetails`) jusqu’aux actions principales (lecture des données, affichage documents, navigation).

### À faire
- Cartographier le parcours d’ouverture de la page (chargement données, états loading/erreur).
- Ajouter un diagramme d’activité (Mermaid) pour le flux principal.
- Ajouter, si besoin, des variantes (erreur chargement, PDF manquant, droits insuffisants).

### Format recommandé
- Mermaid `flowchart` ou `stateDiagram` dans des fichiers `.md` dédiés.
- Un diagramme par fichier pour rester lisible.
