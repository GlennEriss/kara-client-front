## Wireframes – Générer identifiant (V2)

Ce dossier contient les maquettes (wireframes) de la fonctionnalité « Générer identifiant » / réinitialisation du mot de passe.

### Fichiers

- **WIREFRAME_MODAL_GENERER_IDENTIFIANT.md** : modal de confirmation (desktop et mobile), avec champ de recopie du matricule et boutons Annuler / Accepter.

### Contexte

- Le **point d’entrée** est un bouton sur la **liste des membres** (par membre). Le wireframe de la liste est décrit dans `../liste-memberships/` ; ici on se concentre sur le **modal** uniquement.
- Design system : cohérent avec le reste de l’app (shadcn, palette KARA si définie).
