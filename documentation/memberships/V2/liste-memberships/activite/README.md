## Diagrammes d’activité – Liste des membres (V2)

Ce dossier contient les **diagrammes d’activité** (PlantUML) pour la liste des membres :

### Diagrammes disponibles

- **`main.puml`** : Flux principal de chargement de la page liste.
- **`tabs.puml`** : Navigation entre tabs (presets de filtres).
- **`filtres.puml`** : Application de filtres avancés et recherche texte.
