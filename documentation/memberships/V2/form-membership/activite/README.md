## Diagrammes d’activité – Formulaire membre (V2)

Ce dossier contient les **diagrammes d’activité** (PlantUML) pour le formulaire d’ajout/édition de membre :

### Diagrammes disponibles

- **`main.puml`** : flux principal du formulaire (parcours Step1 → Step2 → Step3 → Step4, validation, soumission).
- **`creation-rapide.puml`** : création rapide d’un référentiel depuis le formulaire (ouverture modal, création entité, sélection automatique).
